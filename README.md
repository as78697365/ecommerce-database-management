# E-Commerce Database Management System

This project demonstrates the design of an e-commerce database using MySQL. The database manages customers, products, orders, and payment records.

## Technologies Used

* MySQL
* SQL
* Database Design

## Features

* Customer data management
* Product information storage
* Order tracking
* Payment records

## Files

* ecommerce_db_dump.sql – Contains database schema and sample data.
